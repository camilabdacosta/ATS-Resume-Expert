# ATS-Resume-Expert
Assistente Especializado em Currículos ATS para Desenvolvedores de Software

Este repositório contém os documentos necessários para a configuração de um assistente de Inteligência Artificial especializado em otimização de currículos para desenvolvedores de software, com foco em sistemas ATS (Applicant Tracking Systems).

O projeto foi desenvolvido para ser utilizado na plataforma NotebookLM, transformando um documento de requisitos em uma autoridade técnica capaz de guiar profissionais na melhoria de seus currículos.

# Estrutura do Repositório

O repositório é composto obrigatoriamente por dois arquivos principais:

1.PRD.pdf: Documento de Requisitos do Produto. Contém todo o escopo, funcionalidades, público-alvo e a base de conhecimento que o assistente utiliza para responder.

2.Prompt.pdf: Documento de Engenharia de Prompt. Define a personalidade, o tom de voz e as regras de comportamento do assistente, utilizando técnicas como Zero-Shot e Few-Shot.

# O Assistente: ATS Resume Expert

O assistente foi projetado para atuar como um Especialista Sênior. Suas principais características são:

•
Especialista: Autoridade máxima em currículos para tecnologia.

•
Comunicador: Não apenas responde, mas ensina e fornece exemplos práticos.

•
Integridade: Responde estritamente com base na documentação fornecida, sem inventar informações.

•
Transparência: Reconhece limitações e direciona dúvidas fora de escopo para o suporte oficial.

# Como Configurar no NotebookLM

1.Acesse o NotebookLM.

2.Crie um novo Notebook.

3.Faça o upload do arquivo PRD.pdf na seção de Fontes (Sources).

4.Copie a estrutura do prompt encontrada na seção 6 do arquivo Prompt.pdf.

5.Cole nas Instruções Personalizadas do seu Notebook ou envie como a primeira mensagem no chat.

# Técnicas de Engenharia de Prompt Utilizadas

Para garantir a alta performance e confiabilidade do agente, foram aplicadas:

•
Zero-Shot Prompting: Para garantir que o agente não alucine e se mantenha fiel ao documento original.

•
Few-Shot Prompting: Para padronizar respostas de erro e comportamentos específicos em situações de dúvida.
